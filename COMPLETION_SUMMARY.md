## 🆕 新增功能详细说明

### 1. 文档比较 (Document Comparison)

**功能**:
- 计算两个文档的相似度 (0-1)
- 查找文档之间的差异
- 生成详细的统计信息
- 查找共同短语

**使用示例**:
```python
from AIDocGenius import DocProcessor

processor = DocProcessor()

# 比较两个文档
result = processor.compare_documents("report_v1.txt", "report_v2.txt")

print(f"相似度: {result['similarity']:.2%}")
print(f"差异行数: {len(result['differences'])}")
print(f"长度差异: {result['statistics']['length_diff']} 字符")
```

**应用场景**:
- 版本对比
- 内容去重
- 相似度检测
- 变更追踪

### 2. 文档合并 (Document Merging)

**功能**:
- 合并多个文档
- 智能去重
- 自定义分隔符
- 自动添加文档标题

**使用示例**:
```python
# 普通合并
processor.merge_documents(
    ["chapter1.md", "chapter2.md", "chapter3.md"],
    "book.md"
)

# 智能合并（去重）
processor.merge_documents(
    ["doc1.txt", "doc2.txt", "doc3.txt"],
    "merged.txt",
    smart_merge=True
)
```

**应用场景**:
- 文档整合
- 内容聚合
- 报告生成
- 知识库构建

### 3. 增强的批量处理

**改进**:
- 支持更多操作组合
- 更好的错误处理
- 详细的结果报告
- 性能优化

**使用示例**:
```python
results = processor.batch_process(
    input_dir="documents/",
    output_dir="results/",
    operations=["summarize", "analyze", "translate"]

# 查看结果
```
